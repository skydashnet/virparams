# GenieACS Virtual Parameters

## Acknowledgement

This repository is developed based on the source code from  [EtherGig](https://github.com/EtherGig/genieacs-config)

Several changes, improvements, and additional features have been implemented in this version.

For now, we've added includes support for:
- Huawei (EG8145V5, HG8145V5, etc)
- FiberHome (HG6145D2)
- ZTE (F677V2, F477V2)
- CIOT (MQ220, GM220-S)
- CMCC devices
. Others will follow as they have the devices. This repo will continue to be updated as the situation allows.
